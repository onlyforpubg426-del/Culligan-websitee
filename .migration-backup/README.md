# Culligan Pakistan

Vercel-ready React + Vite website for Culligan Pakistan — a premium water delivery brand in Karachi.

## Deploy to Vercel

1. Import this repo into Vercel
2. Vercel auto-detects settings from `vercel.json`
3. Click Deploy

## Local Development

```bash
pnpm install
pnpm --filter @workspace/culligan-pk run dev
```
